# 错词错句本

> 配合 [每日打卡模板](daily-check-in.md) 和 [周复盘模板](weekly-review.md) 使用。
> 30 天路线第 22-30 天的核心任务就是「整理错词错句」，用这个模板记录并滚动复习。

## 使用方法

1. 每天学习结束后，把当天读错、写错、听不懂的词句记到下面表格。
2. 每条记录写清楚：错在哪、正确形式、下次怎么避免。
3. 每周复盘时把「已掌握」的条目划掉，未掌握的滚动到下一周。
4. 同一条目连续两周复习正确，才算真正掌握。

## 每周滚动复习清单

- [ ] 本周新增错词 ≤ 20 个（贪多嚼不烂）
- [ ] 上周未掌握条目已全部重测
- [ ] 连续两周正确的条目已标记「已掌握」
- [ ] 高频错误类型已写进下周计划（如：重音、时态、介词搭配）
