# English Study

一个面向中文学习者的英语学习资源库。目标不是把链接堆满，而是把真正值得长期使用的资源按场景分类，帮助你更快开始、持续复习、稳定提升。

## 怎么使用

1. 先看 [学习路线](docs/learning-path.md)，按自己的水平选择 30 天计划。
2. 每天固定练一项核心技能：听力、口语、阅读、写作或词汇。
3. 把常用资源加入浏览器书签，避免每天重新找资料。
4. 遇到好资源，按 [贡献指南](CONTRIBUTING.md) 加到对应分类。

## 推荐起步组合

- 听力：`BBC Learning English` + `VOA Learning English`
- 词汇：`Anki` + 一本自己的例句卡片
- 口语：`YouGlish` 查发音 + 每天 10 分钟跟读
- 阅读：`Engoo Daily News` 或分级读物
- 写作：`Grammarly` / `LanguageTool` 辅助修改，再整理错句

## 资源选择标准

- 免费或有可用的免费部分
- 内容稳定、更新较持续
- 能直接用于练习，而不只是方法论
- 适合自学者长期重复使用
- 优先选择开源、官方、社区口碑好的项目
