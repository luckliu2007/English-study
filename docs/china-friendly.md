## 国内环境使用建议

1. 听力资源优先选可下载、可单句循环、有文本的工具。
2. 视频资源优先找带英文字幕的版本，避免只看中文字幕。
3. 海外站点打不开时，不要中断学习，马上切换到国内备用资源。
4. GitHub 项目适合收藏路线、词表、笔记；日常训练放在 App 或本地工具里。
5. 每周固定导出或记录一次生词，避免学习数据只留在单一 App 内。

## 一套更适合国内的日程

## 资源判断标准

- 能不能稳定打开。
- 有没有原文、字幕或例句。
- 能不能下载或离线使用。
- 是否适合每天重复练习。
- 是否能把生词、错句、表达沉淀到自己的笔记里。
