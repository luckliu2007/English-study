## 开源表达

| 场景 | 常用表达 |
| --- | --- |
| 提 issue | `Steps to reproduce`, `Expected behavior`, `Actual behavior` |
| 提 PR | `This PR fixes...`, `Changes include...`, `Testing done...` |
| 请求帮助 | `Could you clarify...`, `I am not sure whether...` |
| 代码评审 | `What do you think about...`, `This might be easier to maintain if...` |

## 练习方法

- 每天读一页英文技术文档。
- 把常见报错和解决方案用英文记录。
- 看开源项目 issue，学习真实工程沟通。
- 写 PR 描述时尽量使用英文模板。
