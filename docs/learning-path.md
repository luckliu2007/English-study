# 学习路线

如果你想让计划更像“系统推荐”，先看 [自适应学习计划](adaptive-plan.md)，再回到本页选择资源和任务。

## 如果你是入门到初级

目标：先建立听得懂、读得下去、敢开口的基础。

### 30 天计划

| 时间 | 任务 |
| --- | --- |
| 每天 10 分钟 | 听 BBC/VOA 慢速英语，先听大意，再看文本 |
| 每天 10 分钟 | Anki 复习高频词，重点看例句 |
| 每天 10 分钟 | 跟读 3-5 句短句，录音回听 |
| 每周 2 次 | 读一篇短新闻或分级读物 |

### 国内推荐工具

| 任务 | 工具 |
| --- | --- |
| 慢速听力 | 每日英语听力 / BBC Learning English |
| 词汇 | Anki / TypeWords / 欧路生词本 |
| 跟读 | 每日英语听力单句循环 / 可可英语 |
| 阅读 | 新概念英语 GitHub 笔记 / Engoo Daily News |

## 如果你是中级

目标：扩大真实语料输入，提高表达自然度。

### 30 天计划

| 时间 | 任务 |
| --- | --- |
| 每天 15 分钟 | 听一段真实播客或 YouTube 英语内容 |
| 每天 10 分钟 | 用 YouGlish 查 3 个短语的真实发音 |
| 每天 10 分钟 | 写 5-8 句英文日记或观点 |
| 每周 1 次 | 整理一篇英文文章的表达和观点 |

### 国内推荐工具

| 任务 | 工具 |
| --- | --- |
| 听力 | 每日英语听力 / 可可英语 / BBC Learning English |
| 口语 | everyone-can-use-english / earthworm / 发音清单 |
| 阅读 | MDN / GitHub Docs / Quanta Magazine |
| 写作 | LanguageTool / DeepL Write / AI 辅助修改 |

## 如果你是进阶

目标：提高专业表达、写作质量和高密度输入能力。

### 30 天计划

| 时间 | 任务 |
| --- | --- |
| 每天 20 分钟 | 听原速播客、访谈、技术演讲 |
| 每天 15 分钟 | 阅读英文长文、技术文档或论文摘要 |
| 每周 2 次 | 写一段 200-300 词英文总结 |
| 每周 1 次 | 做一次 3-5 分钟英文口头复述 |

### 国内推荐工具

| 任务 | 工具 |
| --- | --- |
| 专业输入 | 英文技术文档 / 论文摘要 / 原速播客 |
| 技术表达 | GitHub Docs / English style guide |
| 词汇筛选 | wordfreq / google-10000-english |
| 输出沉淀 | Obsidian / 本地 Markdown / GitHub 仓库 |

## 最小可执行日程

很忙的时候只做这三件事：

1. 听 5 分钟英文。
2. 复习 10 张单词卡。
3. 跟读 3 句话。

持续比强度更重要。

## 国内网络不稳定时的备用流程

1. 海外网页打不开：换到 [中国环境友好资源](china-friendly.md) 里的国内工具。
2. 视频打不开：先用音频和文本完成精听，不等视频。
3. GitHub 打不开：记录项目名，稍后再下载；当天继续用本地模板打卡。
4. 字典打不开：优先使用欧路离线词库或本地生词本。
