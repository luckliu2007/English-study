# 自适应学习计划

这个计划不是复制任何商业产品的内部算法，而是仿照成熟口语学习产品常见的公开设计：先测水平，再按能力点分配任务，根据完成数据动态调整难度。核心目标是让每天学习更像“系统推荐”，而不是临时找资料。

## 设计依据

- CEFR 用 A1-C2 描述语言能力，覆盖听、说、读、写等活动，适合做水平分层。
- 自适应学习的核心是根据学习者表现提供个性化资源和任务。
- 口语产品通常会看发音、流利度、完整度、语义匹配、听力理解等维度。
- 间隔复习适合词汇、短语、句型和易错发音的长期记忆。

## 能力模型

每天记录 6 个能力分，范围 0-5。

| 能力 | 说明 | 主要训练 |
| --- | --- | --- |
| L 听力理解 | 是否听懂主旨和细节 | 精听、泛听、听写 |
| P 发音准确 | 音素、重音、连读、尾音 | 跟读、录音、技术词发音 |
| F 口语流利 | 停顿、语速、连续表达 | 影子跟读、30 秒复述 |
| V 词汇短语 | 高频词、搭配、场景表达 | Anki、TypeWords、例句卡 |
| G 语法句型 | 句子结构和准确性 | earthworm、新概念、改错 |
| W 写作表达 | 清楚表达观点和总结 | 日记、摘要、AI 批改 |

## 入学测评

先完成 [入学测评模板](../templates/placement-test.md)，得到初始等级。

| 起点 | 判断标准 | 推荐路线 |
| --- | --- | --- |
| A1 | 只能听懂慢速短句，输出依赖模板 | 基础发音 + 高频词 + 新概念 1 |
| A2 | 能表达日常需求，但句子短、停顿多 | 慢速听力 + 句型替换 + 新概念 2 |
| B1 | 能谈常见话题，但不够自然 | 真实材料精听 + 复述 + 写短观点 |
| B2 | 能处理多数学习/工作场景 | 原速输入 + 观点表达 + 技术/考试英语 |
| C1 | 能做专业表达，但追求准确和风格 | 长文输入 + 演讲复述 + 高质量写作 |

## 每日任务生成

每天任务由三部分组成，总时长 20-35 分钟。

| 模块 | 占比 | 内容 |
| --- | --- | --- |
| 到期复习 | 25% | 单词、短语、错句、错音 |
| 主线训练 | 50% | 当前等级的听说读写任务 |
| 弱项补偿 | 25% | 上周最低分能力点的专项练习 |

### 任务选择规则

1. 如果昨天完成率低于 60%，今天减少新内容，只做复习和 1 个主线任务。
2. 如果连续 3 天完成率高于 85%，今天增加 1 个输出任务。
3. 如果某能力连续 2 次低于 3 分，明天加入弱项补偿。
4. 如果某能力连续 7 天高于 4 分，下一周提高材料难度。
5. 如果跟读材料连续 2 天低于 3 分，不换材料，继续同一段做精练。

## 每日评分

完成每天任务后，按 0-5 给自己打分。

| 分数 | 含义 |
| --- | --- |
| 5 | 几乎无压力，能稳定复现 |
| 4 | 基本掌握，偶尔卡顿或出错 |
| 3 | 勉强完成，需要看文本或提示 |
| 2 | 很吃力，只能完成一部分 |
| 1 | 基本没听懂或说不出来 |
| 0 | 未完成 |

## 复习间隔

用于单词、短语、句型、错音和错句。

| 评分 | 下次复习 |
| --- | --- |
| 0-1 | 明天 |
| 2 | 2 天后 |
| 3 | 4 天后 |
| 4 | 7 天后 |
| 5 | 14 天后 |

如果同一项目连续两次 4 分以上，升级为“只在周复盘看一次”。如果连续两次 2 分以下，拆成更小单位，例如只练一个音、一个短语或一个句型。

## 等级升级规则

每周复盘一次，不要每天改等级。

| 升级条件 | 操作 |
| --- | --- |
| 4 个以上能力点平均分达到 4 | 下周材料提高半级 |
| 输出任务完成率达到 80% | 增加口语或写作任务 |
| 听力连续 2 周低于 3 | 降低听力材料速度或长度 |
| 发音连续 2 周低于 3 | 减少泛听，增加跟读和录音 |
| 词汇复习堆积超过 80 条 | 暂停新词 2 天 |

## 7 天循环

| 日期 | 重点 | 任务 |
| --- | --- | --- |
| Day 1 | 诊断和输入 | 慢速听力 + 词汇 + 跟读 |
| Day 2 | 发音 | 单句循环 + 录音回听 + 错音记录 |
| Day 3 | 句型 | earthworm / 新概念句型替换 |
| Day 4 | 听力 | 精听 1 段 + 复述 30 秒 |
| Day 5 | 输出 | 写 5-10 句 + 口头复述 |
| Day 6 | 综合 | 读短文 + 摘表达 + 跟读 |
| Day 7 | 复盘 | 周复盘 + 调整下周任务 |

## 90 天成长路线

| 阶段 | 周期 | 目标 | 通过标准 |
| --- | --- | --- | --- |
| 校准期 | 第 1 周 | 找到真实起点和最低分能力 | 完成入学测评和 5 天打卡 |
| 建模期 | 第 2-4 周 | 建立稳定复习和跟读习惯 | 完成率 70% 以上，错词错音开始下降 |
| 提升期 | 第 5-8 周 | 提高输入难度，增加输出 | 每周 2 次 1 分钟复述或 150 词写作 |
| 迁移期 | 第 9-12 周 | 把英语用于真实场景 | 能做一次主题表达、技术总结或考试口语模拟 |

## 难度调节表

| 当前表现 | 材料调整 | 任务调整 |
| --- | --- | --- |
| 听不懂 60% 以上 | 换慢速、短材料、有文本材料 | 减少新词，增加精听 |
| 能听懂但说不出 | 保持材料难度 | 增加复述和句型替换 |
| 单词认识但不会用 | 减少孤立背词 | 增加例句卡和写作任务 |
| 发音一直低分 | 材料不升级 | 单句循环、录音回听、错音复习 |
| 完成率低 | 不升级难度 | 每天只保留 2 个核心任务 |
| 完成率高且分数高 | 提高半级材料 | 增加输出任务 |

## 每天如何开始

1. 打开昨天的打卡记录。
2. 先完成到期复习。
3. 做今天主线任务。
4. 用 0-5 分记录 6 个能力点。
5. 把最低分能力点写进明天的弱项补偿。
