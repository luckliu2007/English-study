# 康奈尔笔记法

> 等级：B1 · 来源：github:luckliu2007/English-study

## 一句话定义

康奈尔笔记法把一页纸分成三区：右侧记要点，左侧写关键词或问题线索，底部写一句话总结，复习时遮住右侧只看线索来回忆。

## 为什么有效

它把「记笔记」和「主动回忆」合二为一。传统笔记只是抄录，复习时容易变成被动重读；康奈尔的左侧线索栏则强迫你事后提炼问题，复习时遮右看左，等于每次翻笔记都在自测。底部的一句话总结又逼你做归纳，加深对整页结构的理解。清晰的分区还让考前速览效率极高。

## 具体步骤

1. 在纸上划出三区：右侧约占 2/3 记正文，左侧 1/3 留白，底部留 2–3 行。
2. 听课或读材料时，只在右侧记要点，用短句和缩写，别逐字抄。
3. 学完后趁热在左侧把每块要点改写成一个关键词或一个问题。
4. 在底部用一句话总结整页内容，作为主题标签。
5. 复习时用手或纸遮住右侧，只看左侧线索凭记忆复述，再挪开核对。

## 英语学习场景实例

看一集英语访谈视频时，右侧随手记下新表达和例句，如 `right`：`I nailed the interview（面试搞定了）`；学完后在左侧写线索「『搞定/成功』怎么说？」，底部总结「本集主题：求职表达」。第二天复习时遮住右侧，只看「『搞定』怎么说」这个问题，逼自己回忆出 `nail it`。这样一页笔记既是记录，也是一套现成的口语自测题。

## 配套练习或模板

选一段 5 分钟的英文视频，用康奈尔三区做一页笔记，然后把左侧线索栏里想不起答案的条目，抄进 [每日打卡模板](../templates/daily-check-in.md) 的「新学表达」表滚动复习。这一方法与 [主动回忆法](active-recall.md) 天然互补——线索栏就是你的回忆触发器。

## 常见误区

- **只记不遮**：如果从不遮住右侧、只是反复通读，就退化成了普通笔记，失去了自测价值。
- **右侧逐字抄录**：把听到的每句话都抄下来会跟不上进度也没重点。右侧只记要点和例句，加工留给左侧和底部。
