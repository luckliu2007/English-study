# 精选合集

这些是 GitHub 上值得收藏的英语或语言学习资源合集，适合继续挖资源。

| 资源 | 说明 |
| --- | --- |
| [yvoronoy/awesome-english](https://github.com/yvoronoy/awesome-english) | 英语学习 awesome list，覆盖听说读写、AI 工具、语法、词汇等 |
| [knowledgefxg/learning-english](https://github.com/knowledgefxg/learning-english) | 中文整理的英语学习资源合集，分类丰富，上手友好 |
| [interaminense/learning-english](https://github.com/interaminense/learning-english) | 经典英语学习链接合集，包含网站、课程、电子书、播客等 |
| [jqhoogland/awesome-language-learning](https://github.com/jqhoogland/awesome-language-learning) | 多语言学习资源合集，可补充学习方法和工具 |
| [Vuizur/awesome-language-learning](https://github.com/Vuizur/awesome-language-learning) | 偏开源工具和语言学习项目，适合喜欢折腾工具的人 |
| [5mdld/anki-english-60k-decks](https://github.com/5mdld/anki-english-60k-decks) | 大型 Anki 英语词汇牌组，包含释义、例句和音频 |

## 使用建议

- 不要一次打开太多合集。先挑一个主资源库，把真正用上的资源搬进自己的学习流程。
- 看到重复推荐的资源，通常优先级更高，例如 BBC Learning English、VOA Learning English、Anki、YouGlish。
