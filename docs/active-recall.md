# 主动回忆法

> 等级：B1 · 来源：github:luckliu2007/English-study

## 一句话定义

主动回忆（active recall）指合上书本、凭记忆把内容主动提取出来（写下来或说出来），再对照原文核对，而不是被动地反复看。

## 为什么有效

记忆的强度取决于「提取」而非「输入」。每一次费力回想，都会强化对应的神经通路，这就是认知科学里的「提取练习效应」（testing effect）。大量研究一致表明，在同样时间下，自测比重复阅读的长期保留率高得多——重读让你「感觉熟悉」，回忆才让你「真正记住」。

## 具体步骤

1. 学完一小节后立刻合上材料，不要马上翻回去。
2. 拿一张空白纸或卡片，凭记忆写下刚才的要点、规则或例句。
3. 卡壳的地方先标个问号，别急着看答案——费力回想本身才是训练。
4. 全部想完后再翻开材料，用不同颜色笔补齐遗漏和错误。
5. 把答错或想不起的条目排进复习计划，按间隔重复再测。

## 英语学习场景实例

学了一组商务邮件高频搭配（如 `look forward to`、`get back to you`、`touch base`）后，合上笔记，只看中文提示「期待/回复你/碰个头」，凭记忆默写出英文表达和一个完整例句。写完再核对，错的当场用红笔改。第二天复习时不看笔记，直接用这几个词各造一句关于今天工作的真句子——这一步把「认得」逼成了「会用」。

## 配套练习或模板

三分钟微练习：读完任意一段英文材料后，盖住原文，用 3 句英文复述大意，写进 [每日打卡模板](../templates/daily-check-in.md) 的「今日复述」区。核对后把想不起的词填进「新学表达」表。到期条目按 [自适应学习计划](adaptive-plan.md) 的复习间隔（0-1 分明天再测，5 分 14 天后）滚动自测。

## 常见误区

- **把「看一遍笔记」当成回忆**：只要眼睛还在材料上，就不是主动回忆。必须先合上，凭空提取。
- **卡壳就立刻看答案**：那一瞬间的「想不起来」正是训练发生的地方，提前看答案会让效果大打折扣。给自己 10-20 秒挣扎时间。
